# AZ104 Projects x5

## Project 1: Azure Compute and Identity Management

## 1. Create Virtual Machine

- Azure Portal → Virtual Machines → Create → Azure VM

- Basics: Resource group (new), VM name, region, Size: Standard B1s (lowest cost)

- Administrator account: Username + password or SSH public key

- Disks: Add data disk → New disk → 32GB → OK

- Networking:
  - Virtual network (default)
  - Create new NSG → Inbound rules: Allow SSH (22) for Linux or RDP (3389) for Windows → No other inbound rules

- Review + Create

## 2. Configure RBAC (Least Privilege)

- Create Entra group:

  - Entra ID → Groups → New group → Security → Name: VM-Admins

  - Add yourself as member

- Assign role to group (after VM deploys):

  - VM → Access control (IAM) → Add role assignment

  - Role: Virtual Machine Contributor → Members: VM-Admins group

- Verify: Sign out/in → Try to delete VM (should fail with permissions error)

## 3. Connect to VM (Test NSG)

- VM → Connect → SSH/RDP

- Note public IP → Try accessing http://<public-ip> in browser (should timeout - NSG blocks port 80/443)

- Connect successfully via SSH/RDP (NSG allows port 22/3389)

## 4. Apply Azure Policy (Tag Enforcement)

- Create custom policy initiative:

  - Policy → Definitions → Initiative definition

  - Name: Require "Environment" tag

  - Add definition: "Require a tag on resources" → Parameters: Tag name = "Environment"

  - Review + create

- Assign initiative:

  - Assign to subscription → Parameters: Tag value not required yet

- Test:

  - Try creating a VM without tag → Should be denied

  - Create VM with tag Environment: Test → Succeeds

- View compliance: Policy → Compliance → Select assignment

## 5. Monitor Costs

- Cost Management → Budgets → Add

  - Name: Monthly limit → Amount: $50

  - Alerts: 50%, 80%, 100% → Email to yourself

- Cost Analysis: View current spend (likely $0 initially)

- Azure Advisor → Cost tab → Review recommendations (e.g., shut down VMs off-hours)

## 6. Clean Up

Delete resource group → All resources removed → No ongoing charges
